## ğŸ› Troubleshooting

### Install Button Not Showing?

**Chrome/Edge:**
- Make sure you're using HTTPS (or localhost)
- Try refreshing the page
- Check if already installed

**Safari iOS:**
- Use the Share button method
- Make sure you're in Safari (not Chrome)

**Firefox:**
- Update to the latest version
- Check the menu for "Install" option

### App Not Working Offline?

1. Make sure you've visited the app while online first
2. Check your browser's storage settings
3. Try clearing cache and reinstalling

### Update Not Applying?

1. Close all tabs with TabScroll open
2. Reopen the app
3. Click "Update Now" when prompted
4. If still not working, uninstall and reinstall
