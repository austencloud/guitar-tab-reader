# TabScroll Scripts

This directory contains utility scripts for project maintenance and development.

## Available Scripts

### `create-icons.js`
Generates project icons and assets for the application.

### `generate-pwa-assets.js` 
Generates Progressive Web App assets including manifests, service workers, and icon variants.

### `update-smart-import.sh`
Shell script for updating smart import functionality and related components.

## Usage

These scripts are typically run during development or build processes. Refer to each script's internal documentation for specific usage instructions.

## NPM Scripts

Many of these scripts can also be executed via npm scripts defined in `package.json`:

```bash
# Generate PWA assets
npm run generate-pwa-assets
```
